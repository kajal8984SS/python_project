🎓 📌 Student Management System – Features
🔐 1. User Authentication (Login System)
Secure login using username & password
Only authorized users can access the system
Session-based authentication
➕ 2. Add Student
Add new student details:
Name
Email
Course
Data stored in database (MongoDB)
📋 3. View Students
Display all students in a table
Clean and structured layout
Real-time data from database
✏️ 4. Edit Student
Update existing student details
Modify name, email, or course
❌ 5. Delete Student
Remove student record from system
Permanent deletion from database
🔍 6. Search Functionality
Search students by:
Name
Email
Course
Uses $regex for flexible searching
📊 7. Dashboard (Analytics)
Total number of students
Course-wise student count
Helps in quick analysis
🎨 8. User Interface (UI)
Built using HTML + Bootstrap
Responsive and user-friendly design
Easy navigation (Upload, Dashboard, etc.)
🗄 9. Database Integration
Uses MongoDB (NoSQL database)
Stores student data in collections
Fast and scalable
🔄 10. CRUD Operations
Create → Add student
Read → View students
Update → Edit student
Delete → Remove student
